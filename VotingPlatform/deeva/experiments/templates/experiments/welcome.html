Welcome to Deeva.

This is a scientific experimental platform using crowd-based knowledge gathering and genetic algorithms for fast generation of virtual characters using non-geometric input.

The platform is going through major changes. The code of the new platform can be found open source at the DeEvA project on GitHub

This new version will generate characters using the Manuel Bastioni Lab add-on for Blender. We are working on a Blender add-on to use the data collected during experiments to generate characters directly inside Blender

Previous work

The previous version of DeEvA used a combination of MakeHuman and custom GUI and scripts. This video shows a working prototype as Published at the Intelligent Virtual Agents - IVA 2017 conference.

The following video describes the original proposal, based on the use of Interactive Genetic Algorithms.
